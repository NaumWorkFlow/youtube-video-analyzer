# Система разбора видео (YouTube) — как это работает

Цель: присылать качественные сводки + критический разбор роликов, как это делал ChatGPT
по промпту. Модель не «смотрит» видео — она берёт **транскрипт (субтитры)** и применяет промпт.

## Как достаётся транскрипт (важно)
В этой среде песочница и web_fetch **не дотягиваются до YouTube** (проверено). Рабочие пути:
- **Сейчас:** расширение **Claude in Chrome** — Claude открывает видео в твоём браузере,
  включает панель субтитров, читает текст, применяет промпт. Требует открытого Chrome.
- **Позже (режим 2):** собственный локальный MCP-сервер для транскриптов — работает без браузера.

## Режим 1 — по запросу (готов)
Пришли ссылку на видео. По умолчанию применяется `PROMPT-general.md`.
- Хочешь другой промпт под конкретное видео — пришли его вместе со ссылкой.
- Можешь задать режим: **«только сводка»** (Фаза 1) или **«полный разбор»** (Фаза 1+2).
- Разбор сохраняется в папку `analyses/`.

## Режим 2 — по расписанию (позже)
Каждое утро при открытом Chrome Claude проверяет каналы из `channels.md`, находит новые
ролики (сверяясь с `seen-videos.md`), применяет общий промпт, складывает разборы в `analyses/`.
Полностью автономный режим 24/7 в этой среде надёжным сделать нельзя — работает, пока браузер открыт.

## Файлы
- `PROMPT-general.md` — общий промпт по умолчанию (v1).
- `PROMPT-biblical-example.md` — пример узкого промпта под тему; шли вручную под библейские видео.
- `channels.md` — список каналов для мониторинга (режим 2).
- `seen-videos.md` — лог уже разобранных видео (режим 2).
- `analyses/` — готовые разборы.
